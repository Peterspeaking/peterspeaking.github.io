IPv4 Subnetting Helper “Wall” Method

Step-by-step like the whiteboard: convert to binary → build the subnet mask (prefix = 1s) → mark the wall → flip host bits for Network ID / Broadcast → use host bits for totals → add the block size for Next Subnet.

Try it



| Wall = split (network | host)
Bit weights (per octet): 128 64 32 16 8 4 2 1
Each IPv4 address has 4 octets × 8 bits = 32 bits total.

Common mistakes

Prefix confusion: /26 means 26 network bits, not “26 hosts.”

Binary grouping: keep octets as groups of 8 bits. The wall can split inside an octet.

Next subnet: don’t add 1 to the IP address; add the block size to the network ID.
